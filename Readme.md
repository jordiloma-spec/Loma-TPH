# Tu / Jo – Joc deductiu musical

Un joc interactiu basat en una cançó.

Escolta, observa i dedueix:
qui és el **Tu** i qui és el **Jo**.

## 🎮 Com es juga
- Mira el videoclip
- Selecciona només les frases que donen pistes
- Les frases genèriques no ajuden (i poden fer-te recular)
- El temps juga un paper important

## 🎵 Mecànica
Cada frase-pista elimina possibles identitats.
Quan la cançó s’acaba:
- si només queda un nom per a TU i un per a JO → has resolt el misteri
- si no → torna-ho a intentar

## 🕵️‍♀️ Inspiració
Un joc entre la lògica, la poesia i l’amor.
